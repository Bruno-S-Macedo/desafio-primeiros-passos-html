Tags de Textos

(back)

Inicialmente foi demonstrado o uso das tags de titulo e subtitulos, ou seja, o uso das tags <h1>, <h2>, <h3>, <h4>, <h5> e <h6> . o primeiro nivel é caracterizado como o titulo principal, o segundo nivel é definido como o sub titulo do primeiro nivel, o terceiro como sub titulo do segundo e assim por diante. Outra tag relacionada com textos é a tag <strong> ela é utilizada para tornar o conteúdo dentro dela em negrito, tornar o conteúdo forte se levar pro literal. No HTML , há uma tag exclusiva para abreviações, a tag utilizada tem como sintaxe <abbr> ela traz o recurso de usar abreviações e ainda ter como opção ela escrita por extenso ao passar o cursor do mouse em cima da mesma.

Ademais também foi utilizado a tag para criar paragrafos, que tem como sintaxe <p>, essa tag tem como função delimitar parágrafos, como por exemplo esse que que você está lendo neste momento. Tradicionamente utilizamos marca-textos para dá um destaque maior em palavras-chaves, no HTML não é diferente, é utilizada a tag <mark> para fazer uma marcação. Outra tag usada em textos é a tag <small> que é usada para tornar o conteúdo dentro dela com uma fonte pequena. Ainda relacionada com a forma dos textos, tem a tag <blockquote> utilizada para delimitar citações.

Assim como há uma tag para tornar textos negrito, há uma tag para tornar Textos Italicos, a tag utilizada é a <i>. T ambém há uma tag para Textos Sublinhados, a tag <u>. Além dessas citadas nesse Artigo, há várias outras tags relacionadas a textos.

Tags de Listas

(back)

  • Lista Ordenadas

Listas ordenadas são listas na qual a ordem tem importancia, um exemplo de lista ordenada é uma receita, não se pode começar pelo ultimo ingrediente, ou seja, a ordem importa. A tag utilizada para delimitar uma lista ordenada é <ol>, alem dessa tag é utilizada outra para cada item dessa lista, no caso a tag <li>

  • Lista Não Ordenadas

Ao contrario do que a Lista Ordenada, as Listas Não Ordenadas são listas onde a ordem não tem importancia, também são chamadas de listas de marcadores. A tag utilizada para delimitar essse tipo de lista é <ul>, assim como na outra, a tag utilizada para cada item da lista é <li>.

Tags de Links

(back)

O link é a essencia fundalmental do HiperTexto. Ele nos permite ligar documentos entre si e permitir a navegação entre essas páginas. Para criar um hyperlink, devemos criar âncoras atráves da tag <a>. Nessa tag, o atributo principal é o href que cria uma referência Hipertexto.
